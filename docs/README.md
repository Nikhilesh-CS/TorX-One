# TorX One Documentation

This folder contains the public documentation for TorX One.

TorX One is a privacy-first Android messenger designed for decentralized communication through Tor hidden services and nearby mesh transport.

## Current status

TorX One is beta software. It is suitable for demonstrations, controlled testing, technical review, and early user feedback. It should not yet be treated as audited high-risk operational-security software.
