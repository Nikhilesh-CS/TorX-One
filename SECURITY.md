# Security Policy

## Supported Versions

Currently, only the latest release of TorX One is actively supported with security updates. 

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Because TorX One is a privacy-preserving and security-focused application, we take vulnerabilities extremely seriously.

If you discover a security vulnerability in TorX One, please **do not** open a public issue. Instead, please report it via private communication directly to the maintainers at the associated GitHub email.

We will acknowledge receipt of your vulnerability report as soon as possible, and strive to send you regular updates about our progress. If you report a vulnerability that affects the Tor daemon itself, we may coordinate with the upstream Guardian Project and The Tor Project to address the issue.
